ROS 2 Cheatsheet: Essential Commands and Concepts

This cheatsheet provides a quick reference for the most commonly used ROS 2 commands, concepts, and development practices. It's designed to be a handy guide for both beginners and experienced ROS 2 developers.

1. Workspace Management​

CommandDescription
mkdir -p ~/ros2_ws/srcCreate a new ROS 2 workspace
cd ~/ros2_wsNavigate to the workspace root
colcon buildBuild packages in the workspace
colcon build --packages-select <pkg>Build specific packages
colcon build --symlink-installBuild with symlinks (useful for Python development)
source install/setup.bashSource the workspace (make packages available)
echo "source ~/ros2_ws/install/setup.bash" >> ~/.bashrcAdd sourcing to .bashrc for persistence

2. ROS 2 Node Management​

CommandDescription
ros2 run <package_name> <executable>Run an executable from a package
ros2 node listList all active ROS 2 nodes
ros2 node info <node_name>Display information about a specific node

3. ROS 2 Topic Commands​

CommandDescription
ros2 topic listList all active topics
ros2 topic list -tList topics with their types
ros2 topic info <topic_name>Display information about a topic (type, publishers, subscribers)
ros2 topic echo <topic_name>Print messages published on a topic
ros2 topic pub <topic> <type> <args>Publish data to a topic (one-time)
ros2 interface show <msg_type>Show the definition of a message type

4. ROS 2 Service Commands​

CommandDescription
ros2 service listList all available services
ros2 service list -tList services with their types
ros2 service info <service_name>Display information about a service
ros2 service call <service> <type> <args>Call a service with arguments
ros2 interface show <srv_type>Show the definition of a service type

5. ROS 2 Action Commands​

CommandDescription
ros2 action listList all available actions
ros2 action list -tList actions with their types
ros2 action info <action_name>Display information about an action
ros2 action send_goal <action> <type> <goal>Send a goal to an action server
ros2 interface show <action_type>Show the definition of an action type

6. ROS 2 Parameter Commands​

CommandDescription
ros2 param listList all parameters on all nodes
ros2 param list <node_name>List parameters for a specific node
ros2 param get <node> <param>Get the value of a parameter
ros2 param set <node> <param> <value>Set the value of a parameter
ros2 param dump <node_name>Dump all parameters of a node to a YAML file

7. ROS 2 Launch Commands​

CommandDescription
ros2 launch <package> <launch_file>Run a launch file

8. Debugging and Introspection​

CommandDescription
rqt_graphVisualize the ROS 2 computation graph (nodes and topics)
rviz23D visualization tool for robot state and sensor data
ros2 bag record -aRecord all topics to a bag file
ros2 bag play <bag_file>Play back a recorded bag file

This cheatsheet covers the most frequently used commands. For more detailed information, always refer to the official ROS 2 documentation.
